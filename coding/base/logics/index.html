Conditions logiques

Les conditions

Une condition permet d'exécuter différentes actions en fonction d'une valeur. Elles se trouvent dans l'onglet Logique de MakeCode. Elles sont très utilisée en programmation. Elles permettent à l'ordinateur de faire un choix et de prendre un chemin différent.

Vrai ou faux

Une condition est toujours vraie ou fausse. Si elle est vraie, alors les blocs contenus dans la condition sont exécutés. Si elle est fausse, alors les blocs sont ignorés.

Comparer des valeurs

Les valeurs par défaut dans les conditions ne sont pas très utiles car elles ne changent pas. On peut par exemple remplacer ces valeurs par une variable. En programmation, une variable qui vaut 0 est fausse et vraie si elle vaut 1.

Une condition peut également exécuter des actions si la valeur est fausse. Pour cela, on peut ajouter une case Sinon en appuyant sur le bouton + en bas de la condition.

Comparer une variable avec une valeur

Toujours dans l'onglet Logique, se trouve une catégorie de Comparaison. Ces blocs, un peu spéciaux, sont utilisés dans les conditions. Ils permettent de comparer deux valeurs ensemble. On peut ainsi déterminer si une valeur est égale, supérieur ou inférieure à une autre.
